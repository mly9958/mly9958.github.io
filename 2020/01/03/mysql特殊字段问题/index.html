Mysql中数据表特殊字段问题

发表于 2020-01-03 分类于 mysql 阅读次数： Valine：
本文字数： 148 阅读时长 ≈ 1 分钟

最近在工作写代码的时候，使用springdatajpa向数据库插入数据时，怎么都插入不进去，一直报语法错误，我检查了字段类型，字段名称，都没有问题。

最后在网上查阅资料时发现，我数据库表有个字段名称为desc，这个名称和mysql数据库关键字冲突了，导致一直报错，更改为description后正常
